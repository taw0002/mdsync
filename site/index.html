Open source markdown viewer and editor

Turn any markdown file into a beautiful, editable document.

Run mdsync README.md in your terminal. It opens in the browser: beautifully rendered, ready to edit inline, with review and comments built in.

Try it right here View on GitHub
Install
npx mdsync README.md
How it works
Your .md file, upgraded in one command.

Start with a markdown file or a whole docs/ folder. Run mdsync. Get a polished browser document with typography, code blocks, inline editing, and review tools without changing your source of truth.

1. Markdown file 2. Run mdsync 3. Edit in browser

Render any markdown

Point it at any .md file or a whole docs/ folder. Get beautiful typography, code highlighting, tables, and dark or light themes.

Edit inline

Double-click to edit in place and save changes back to the original markdown file. No copy-paste between editor and preview.

Review & collaborate

Approve, reject, or comment on any section with a built-in feedback panel designed for async review workflows.

AI-ready (MCP server)

Works with Claude, Cursor, VS Code, and any MCP-compatible tool so agents can read and edit your docs programmatically.

Use cases

Use it for docs, notes, and AI-generated markdown.

Repo docs

mdsync ./docs gives you an instant documentation browser for any project.

Personal notes

mdsync ~/notes is a lightweight Obsidian alternative built on plain markdown files.

AI workflows

AI agents already output markdown. mdsync makes that output readable, editable, and reviewable.

Static sites (coming soon)

mdsync build ./docs will generate a deployable docs site from the same markdown source.
